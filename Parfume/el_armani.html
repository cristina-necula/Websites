Armani

Fondatorul Casei Armani este Giorgio Armani. El s-a născut în 1934 într-o familie săracă din Piacenza, Milano. În 1975, a fost înfiinţată casa de modă Armani şi, odată cu aceasta, sacoul pentru femei – creat din ţesăturile tradiţionale ale sacoului bărbătesc, având imprimată autoritatea masculină.

Trecerea de la moda haute couture la parfumuri a fost aproape naturale. In 1975 lanseaza primul sau parfum, Azzaro Couture, senzual, elegant, din familia mirosurilor ciprate, cu tente floral-fructate, direct inspirat din tendintele vestimentare Loris Azzaro. 1978 a fost anul Azzaro pour Homme, pe care Loris Azzaro l-a conceput ca pe un adevarat autoportret: seducator si charismatic.

Acqua di Gio

Acqua di Gio de Giorgio Armani este un parfum Aromatic Acvatic de barbati. Acqua di Gio a fost lansat in 1996. Creatorul acestui parfum este Alberto Morillas.

Notele de varf: portocala, lamaie verde, mandarina, iasomie, bergamot, lamaie si neroli
Notele de mijloc: ciclama, nucsoara, mignonette, coriandru, violeta, frezie, note marine, piersica, zambila, trandafir, iasomie, rozmarin si calona
Notele de baza: ambra, paciuli, lichen, cedru si mosc alb

Armani Attitude

Armani Attitude de Giorgio Armani este un parfum Oriental Fougere de barbati. Armani Attitude a fost lansat in 2007. Armani Attitude a fost creat de Annick Menardo, Alberto Morillas si Olivier Cresp. Acest parfum este castigatorul premiului FiFi Awards Best Packaging Men`s Prestige 2008.

Notele de varf: cafea si lamaie siciliana
Notele de mijloc: lavanda si cardamom
Notele de baza: paciuli, ambra, opoponax si cedru de China

Armani Eau Pour Homme

Armani Eau Pour Homme de Giorgio Armani este un parfum Citric Aromatic de barbati. Armani Eau Pour Homme a fost lansat in 1984. Creatorul acestui parfum este Roger Pellegrino.

Note de top: mandarina, portocala verde, busuioc, bergamot si lamaie
Note de mijloc: ucsoara, garoafa, crin, scortisoara, lavanda, iasomie si trandafir
Note baza: lemn de santal, paciuli, mosc, lichen, vetiver si cedru
