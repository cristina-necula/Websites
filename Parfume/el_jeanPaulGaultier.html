Jean Paul Gaultier

Jean Paul-Gaultier s-a nascut pe 24 aprilie 1952 in Arcueil, Val-de-Marne, Franta si este printre cei mai celebri si controversati designeri francezi din lume. Inca de pe vremea cand era doar un copil, acesta si-a manifestat pasiunea pentru desen si pentru haine, petrecand timpul alaturi de bunica sa.

In anul 1993, Jean Paul Gaultier si-a facut debutul pe piata parfumurilor cu parfumul „Classique”, un parfum pentru femei la realizarea caruia s-au folosit esente florare de inspiratie orientala. Parfumul a fost urmat doi ani mai tarziu de catre un parfum pentru barbati „Le Male”. Ambele au fost extrem de bine primite iar parfumul „Le Male” este cel mai popular parfum barbatesc din Uniunea Europeana.

Le Male

Le Male de Jean Paul Gaultier este un parfum Oriental Fougere de barbati. Le Male a fost lansat in 1995. Creatorul acestui parfum este Francis Kurkdjian.

Notele de varf: pelin, lavanda, menta, bergamot si cardamom
Notele de mijloc: chimion, floare de portocal si scortisoara
Notele de baza: lemn de santal, boabe de tonka, ambra, vanilie si cedru

Le Beau Male

Jean Paul Gaultier Le Beau Male este o apă de toaletă din categoria aromatic-fougere, extrem de fresh care evidenţiază senzualitatea bărbătească. Intr-un cuvânt este un Apollo care face pe toată lumea să se topească după aroma sa. Prin compoziţia sa memorabilă şi mixul ingredientelor, acest parfum oferă o gamă unică de note puternice şi reci ca gheaţa.

Notele de varf: Mentă, Pelin
Notele de mijloc: Lavandă, Floare de portocal, salvie
Notele de baza: Mosc

Le Male Terrible

Le Male Terrible de Jean Paul Gaultier este un parfum Oriental Fougere de barbati. Le Male Terrible a fost lansat in 2010. Creatorul acestui parfum este Aurelien Guichard.

Note de top: grapefruit si piper roz
Note de mijloc: lavanda si vetiver
Note baza: ambra si vanilie
