Lancôme

Lancome a luat fiinta in Franta in anul 1935, fondatorul ei fiind Armand Petitjean.

In urma vizitei fondatorului la ruinele castelului Le Chateau de Lancosme a fost creat brandul. Simbolul Casei Lancome, trandafirul, a fost ales in aceeasi imprejurare. Lancôme este simbolul frumusetii frantuzesti. Istoria Lancôme este strans legata de moda, frumusete si eleganta. Primele parfumuri marca Lancome au fost lansate in acelasi an 1935 – Tendre Nuit, Bocages, Conquete, Kypre si Tropiques. Toate cele cinci parfumuri sunt in prezent parfumuri de colectie.

Lancôme Tresor in Love

Lancome a lansat Tresor in Love in Martie 2010 fiind creat de parfumierii Dominique Ropion si Veronique Nyber. Esenta noului parfum incorporeaza note fructat-florale in cadrul unei baze lemnoase si de cedru. Tresor in Love imbie prin acordurile de inceput ale bergamotei, perelor, nectarinelor si ale piperului. Flaconul este lung si rozaliu, decorat cu un trandafir negru satinat in jurul gatului. Lancome Tresor in Love este un parfum modern care evidentiaza momentul primei dragoste.

Lansat: 2010
Parfumier: Dominique Ropion si Veronique Nyber
Notele de top: pere, nectarine, piper, bergamota
Note de mijloc: trandafir, piersici, violete, iasomie
Note de bază: mosc, cedru

Lancôme Tresor Midnight Rose

Lancome Tresor Midnight Rose este un parfum plin de culoare, de jovialitate si de veselie… Intruchipeaza feminitatea fara astampar: trandafirul devine jucaus, animat de tonuri delicios de intepatoare de zmeura, licariri lemnoase si note voluptoase de vanilie. Aceasta roza irezistibila si spontana vrajeste totul in calea ei

Lancome Trésor Midnight Rose prezinta un buchet floral catifelat, cu accente de lemn, adecvat in special pentru zilele reci. Esentele cheie ale acestui parfum de lux sunt dulci, fructate si bogat florale. Parfumul Tresor Midnight Rose este senzual și de dorit, nici un om nu-i poate rezista. Este catifelat, dulce si exotic. Trésor Midnight Rose este accesoriul perfect pentru o femeie care vrea să fie în centrul atenției, este increzatoare si misterioasa. Este parfumul perfect pentru orice ocazie speciala sau eveniment.

O nouă doză de romantism parizian, parfumul incepe cu acorduri fine de zmeură. Dragoste la prima vedere care s-a întâmplat între petale de trandafir delicat si coaja de fructe rosii de zmeura continua cu un indiciu de bujor si iasomie, într-un mod elegant si sofisticat. In notele de bază gasim mirosul de vanilie calda si moale, nuante lemnoase de cedru şi un voal de mosc senzual.

Lancome La Vie Est Belle

Lancome La Vie Est Belle este un parfum floral fructat gurmand de dama. La Vie Est Belle a fost creat de Olivier Polge, Dominique Ropion si Anne Flipo si va fi disponibil pe piata incepand cu toamna lui 2012. Conceptul acestui parfum este centrat pe ideea de frumusete naturala si simpla, libertatea de convenții și alegerea de viziune dată propria fericire. Parfumul este un fel de perspectivă asupra vieții, inspirat de bucurie și plăcere în lucrurile mici.

Lansat: 2012
Parfumier: Olivier Polge, Dominique Ropion si Anne Flipo
Note de top: para, coacaze negre
Note de mijloc: iris, iasomie, floare de portocal
Note baza: patchouli, vanilie, boabe de tonka, pralina
